# Master Degree Works

## 项目简介

本项目是硕士学位期间的研究工作成果，包含多个子模块，涵盖了从数据采集、后端服务到核心算法验证的完整技术链路。

## 目录结构

- `apps/triple-helix/` - 核心应用/实验模块（主要基于 Objective-C++）
- `flask/` - 基于 Flask 的 Web 后端服务
- `nlps/` - 自然语言处理相关模块
- `spiders/` - 网络爬虫（包含知网爬虫等数据采集工具）

## 技术栈

- **主要语言**: Objective-C++, Python, JavaScript
- **后端框架**: Flask
- **其他**: HTML, Less, CSS
